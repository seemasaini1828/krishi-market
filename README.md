# 🌾 Krishi Market

A Full Stack Farmer-to-Consumer Agri Marketplace built using the MERN Stack.

## 📌 Project Overview

Krishi Market is an online marketplace that directly connects farmers with customers. Farmers can sell their products online, customers can place orders, and admins can manage users and products.

## 🚀 Features

### Customer

* User Registration & Login
* Browse Products
* Add to Cart
* Checkout
* Order History

### Farmer

* Farmer Registration
* Add Products
* View My Products
* Delete Products

### Admin

* View All Users
* View All Products
* Delete Users
* Delete Products
* Dashboard Statistics

## 🛠 Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas
* Mongoose

### Authentication

* JWT (JSON Web Token)
* bcrypt.js
