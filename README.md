🌾 Krishi Market - Farmer to Consumer Platform

📌 Overview
Krishi Market is a full-stack web application that connects farmers directly with consumers.

🚀 Features

👨‍🌾 Farmer:
- Add Products
- View Products
- Delete Products

🛒 Customer:
- Browse Products
- Add to Cart
- Place Orders

👨‍💼 Admin:
- View Users
- View Products

🛠 Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express.js  
Database: MongoDB  
Auth: JWT
